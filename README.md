**To-Do List App**

A simple and responsive To-Do List web application built with HTML, CSS, and JavaScript. Users can add, edit, delete, and mark tasks as complete. Tasks are saved in localStorage for persistence.

**Features:**

Add new tasks

Edit existing tasks

Delete tasks

Mark tasks as complete/incomplete

Data stored in localStorage

Clean UI with light theme

SVG icons for actions

**Tech Stack**

HTML5

CSS3 (Responsive, modern layout)

Vanilla JavaScript (DOM, localStorage)
